<파이썬 Tool 설치하기>

  1. 웹 브라우저로 아래 파이썬 웹사이트에 접속한다.
    https://www/python.org


  2. Download 탭을 선택하고 가장 최신 버전을 다운 받는다.
  3. 내려 받은 python-3.x.x 파일을 더블클릭해서 실행한 다음 "Add python 3.6 to PATH"를 체크한다.
  4. Install Now를 클릭한다.
    --> 프로그램이 설치된다.
  5. "Setup was successful" 화면이 나타나면 설치가 끝난 것
    --> Close 버튼을 눌러 설치를 끝낸다.
